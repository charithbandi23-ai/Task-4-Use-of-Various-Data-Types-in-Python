4.4 Word Frequency Count

Scenario: Write a program to count the frequency of each word in a paragraph.

Algorithm:
1. Input the paragraph.
2. Split the paragraph into words.
3. Count the frequency of each word using a dictionary.

Code:
paragraph = "Python is great. Python is easy to learn. Learn Python and be productive."
word_count = {}
for word in paragraph.split():
    word = word.lower().strip('. ,')
    if word in word_count:
        word_count[word] += 1
    else:
        word_count[word] = 1
print("Word frequencies in the paragraph:")
for word, count in word_count.items():
    print(f"{word}: {count}")
